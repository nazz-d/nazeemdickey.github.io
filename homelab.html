Proxmox + OPNsense Homelab

Designed and deployed a virtualized homelab environment using Proxmox VE and OPNsense to practice real-world networking, firewalling, virtualization, backups, secure remote access, and system administration.

Proxmox VE OPNsense Cisco Switching Synology NAS Docker LXC

Logical Network Diagram

This diagram shows the high-level design of my homelab without exposing private IP addresses, credentials, serial numbers, or sensitive network details.

What I Built

  • Configured OPNsense as the primary edge firewall and router.
  • Connected Cisco managed switches for wired and PoE devices.
  • Deployed multiple Proxmox servers for VMs, LXC containers, and Docker services.
  • Integrated NAS storage and backup infrastructure for lab reliability.
  • Documented a logical layout showing core devices, services, and traffic flow.

Skills Demonstrated

  • Network topology planning
  • DHCP, DNS, and internal routing
  • Firewall rule management
  • Virtual machine and container deployment
  • Secure remote access design
  • Infrastructure documentation

Technologies Used

Tools and platforms used in the lab environment.

Proxmox VE OPNsense Linux Windows VMs Docker LXC Containers Cisco Switches Synology NAS Cloudflared UniFi AP

Security Notes

I avoid exposing sensitive management ports directly to the public internet. Remote access is handled through controlled access methods and firewall rules. Public documentation is sanitized to avoid leaking private IP addresses, public IPs, credentials, or device-specific identifiers.
